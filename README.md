# 音频质量分析器 Audio Analyzer Pro v7.0

**双击 HTML 即可使用 · 完全离线 · 无需安装 · 支持 20+ 音频格式**

---

## 这是什么

一个单文件浏览器端音频质量分析工具。拖入音频文件，自动输出：

- **频谱图** — 全频段能量分布可视化
- **频率响应** — 各频段平均能量
- **相位分析** — 立体声声道相位关系
- **LUFS 响度** — ITU-R BS.1770-4 广播标准（Integrated / Short-term / LRA）
- **True Peak** — 4x 过采样真实峰值检测
- **动态范围** — 峰值/谷值/平均值分析
- **削波检测** — 采样级削波计数
- **DC 偏移** — 直流偏移检测
- **比特深度分析** — 有效比特深度验证（识别假无损）
- **智能诊断** — 12 条规则自动推断母带问题（响度战争、软削波、伪高解析度等）
- **中文大白话解读** — 不需要专业知识也能看懂

## 使用方法

1. 用浏览器（Chrome / Edge / Firefox）打开 `音频质量分析器.html`
2. 拖入一个或多个音频文件到页面
3. 等待分析完成，查看结果

## 技术亮点

- **纯 JS ALAC 解码器** — 从 Apple 开源的 C 源码移植，包含 Rice/Golomb 熵编码、自适应线性预测、20 种预测模式
- **ISO BMFF 解析器** — 单次遍历 MP4/M4A box 树，提取 ALAC magic cookie
- **ITU-R BS.1770-4** — K-weighting + 双门限 LUFS 集成响度，广播级精度
- **4x True Peak** — Catmull-Rom 插值过采样，采样间过载检测
- **HiDPI Canvas** — 频谱图 / 响度曲线 / 相位示波器全部支持高分辨率屏幕
